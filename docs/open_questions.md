## Q1. HTMLスライドのデザイン統一 vs 記事別変更

**質問：**
HTMLスライドのデザインは、記事ごとに変えたいですか？それとも統一テンプレートでOKですか？

| 選択肢 | 内容 | 影響 |
|--------|------|------|
| A：統一テンプレート | 全記事・全画像で同じデザイン | シンプル実装。MVP向き |
| B：記事テーマに合わせて自動変更 | AIが記事トーンに合ったテーマを選択 | テーマ管理が必要。少し複雑 |
| C：ユーザーが手動選択 | テーマ一覧からUIで選ぶ | UI実装が必要 |

**影響範囲：** [F-04 HTMLスライド生成]

## Q2. ユーザー確認ステップの有無

## Q3. 最終出力形式

**質問：**
最終的な出力は「PNG画像のZIPダウンロード」でよいですか？
MarkdownやHTMLへの自動埋め込みも欲しいですか？

| 選択肢 | 内容 | 影響 |
|--------|------|------|
| A：PNG の ZIP のみ | 画像ファイル + 挿入位置マップ JSON | シンプル。MVP向き |
| B：ZIP + 埋め込み済み Markdown | 元テキストに `![](image_01.png)` 挿入済み | Markdown 処理が必要 |
| C：ZIP + 埋め込み済み HTML | 元テキストに `<img>` タグ挿入済み | HTML 処理が必要 |
| D：A + B（両方） | ZIP の中に画像・JSON・Markdown をすべて含む | 実装コスト増だが便利 |

**影響範囲：** [F-06 出力仕様]

フローチャートや比較表などのコンテンツタイプは、AIに自動判断させますか？

| 選択肢 | 内容 | 影響 |

| A：AI 全自動 | AIが内容を読んで最適なタイプを選択 | シンプル。MVP向き |

| C：ユーザーが先に指定 | 生成前にタイプを選んで指示 | UX複雑化 |
